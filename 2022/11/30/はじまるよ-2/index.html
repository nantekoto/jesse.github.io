はじまるよ~博客搭建日志 2

2022年11月30日 博客搭建 约640字 预计需要2分钟

今天我平泽唯就要再给你一砣儿

代码高亮

似乎在默认配置文件里关闭prismjs就好了？

但是又发现主题的代码框第一行存在一个空格的缩进，应该是个bug吧，提了issue希望能被修复。

social模块

虽然不太清楚原理，但是瞎搞弄成了

主题作者有写自己弄图标替换的教程，但是不是太详细：

  • 使用阿里的图标库:登陆阿里矢量库注册 (iconfont网站)
  • 图标管理
  • 新建项目
  • 编辑项目页面 Font Family 参数修改为nexmoefont (似乎是继承的意思，不懂咕噜)
  • 添加新图标到项目 (只需要添加想要的图标就行了，google一个svg或者自己画)
  • 生成的 css 地址添加到主题配置文件 iconlib 参数的 value 部分 (默认是空，就自己加个)
  • 在使用图标部分引用在矢量库的 class （很有问题，是修改引用的类吗？不太懂，反正我是直接改的引用的名字）

反正只需要添加想要改的或者想加的新图标进自己的项目就够了，应该是继承了nexmoefont就能两个图标库共存。

然后在引用里改成自己上传图标的名字，注意，引用的是 icon- 再接上图标名字

图标为github-replace

引用icon-github-replace

bangumi的图标暂时没找到svg，再说。最后注意每次更新图标库，css地址都要发生变化，需要重新配置

Task list

  • 代码高亮：有点问题，暂且蒙古里（虽然我并一定会写很多代码
  • social模块：
    • 这个主题我唯一想换掉的就是左下角github的图标，太憨憨了（文档提供了方法
    • 一些额外的个人链接想要加上，要摸索下
  • 访问统计：在加了在加了，虽然只有自己刷刷数据（悲）。
  • 本地搜索：可能会加？也可能懒得弄了
  • 网站索引：要去google和baidu上登记一下（争取一点微弱的曝光量
  • 关于博主：可能这个模块才是比较重要的吧，有点懒，想写成CV的样子又觉得该随意点
  • 个人域名：又要花钱，花钱的事最后再说
  • Latex支持：latex公式写着很有意思，虽然我肯定不会在博客写，但说不定会装上
